#  Sales Performance Analysis & Forecasting

##  Problem Statement
Businesses struggle to convert raw sales data into actionable insights for strategic decision-making.
Traditional reports lack interactivity and predictive capabilities.
This project builds an AI-driven dashboard to analyze historical sales data and forecast future sales trends.


##  Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Power BI


##  Dataset
- Superstore Sales Dataset (5,000 records)
- Includes sales, profit, region, category, discount, and time-based features


##  Project Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Machine Learning (Sales Forecasting)
5. Power BI Dashboard Development

##  Machine Learning
- Models Used: Linear Regression, Random Forest Regressor
- Evaluation Metrics: MAE, RÂ²
- Final Model: Linear Regression


##  Power BI Dashboard
- Executive KPIs (Sales, Profit, Orders)
- Regional & Category Performance
- Top Products & Loss Analysis
- Actual vs Predicted Sales Forecast


##  Key Insights
- Identified top-performing regions and categories
- High discounts negatively impact profitability
- Seasonal sales patterns detected
- ML model successfully predicts future sales trends
