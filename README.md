# sales-performance-analysis

Performed comprehensive data cleaning including data type correction, missing value handling, duplicate removal, and business rule validation to ensure high-quality sales data for analysis and modeling.

**EDA**

Performed exploratory data analysis to uncover sales trends, regional performance, category-wise contribution, and the impact of discounting on profitability using Matplotlib and Seaborn.

**Feature Engineering**

Engineered time-based features and aggregated monthly sales data to prepare a machine learning-ready dataset for sales forecasting.
